Inergo-OS

Doing your homework in ARCH linux is cool!

What is Inergo-OS?

Inergo-OS is an Arch linux based distro aimed towards students. It works out of the box but has A LOT of customization options.

DE Customs

The custom desktop environment(which uses i3-gaps) is innovative. We give you lot's of options for customization. This allows you to make the perfect customized workspace. You got background, transparency, color schemes, visualizers, clocks, etc.

U need? U get!

We made an easy to use software installer, connected to a lot of handy repositories. Yet the distro comes with minimum software. If you need it, you can find it on here. It contains software for everything a student needs. Music Daws, renderers, editors, etc.

Easy support!

Inergo OS is based on Arch linux and most of the kernel has been left untouched. If ever anything goes wrong, it is very easy to look up your problem. We have a forum tab for distro related issues, but most errors you will get are universal arch linux issues. Google is your best friend.

Productivity

Comes with various handy productivity software such as openlib(which is basically a Microsoft Office clone), thundra and more. It also comes with a fully fleshed out installation of wine, which allows you to use most windows software.

Games & Fun

You can't be studying ALL the time, every time. Seeing as the steam deck is running arch-linux, there is no reason you shouldn't be able to play games on your arch device. And you're right. Inergo comes with steam(if you enable it in the installer) as well as discord
